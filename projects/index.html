PROJECTS

Projects I found interesting enough to save.

Dance Dance Revolution Dedicab »
Deconstructed, cleaned, and revamped old JP DDR cab. Emulated analog experience with Ubuntu Studio for its low-latency kernel and original, troublesome CRT for high refresh rates.

Sound Voltex Controller »
Cheap DIY controller for a rhythm game. Used Arduino Micro, some arcade buttons, and a cardboard box.

Apache Spark Clusters »
Constructed a cluster-computing framework using Apache Spark in Python3. Two personal computers are slightly faster than one.

Image Stitching »
Learned basic image stitching procedures using Python3 and OpenCV. Enabled fun planar picture profiling.

32-bit Processor Design »
Designed and synthesised a 32-bit processor using a base 8-bit CPU with Virtuoso.
Approached again with ModelSim, except from scratch.

Scarlet Hawks UAV »
Help design the image processing for obstacle avoidance.
2017 AUVSI SUAS competition was fun, but we certainly had a lot to learn.

CPU Monitor »
Made a CPU monitor UI shell programmed using assembly (MIPS). Really makes you appreciate high level abstraction.

LiCoO2 Batteries »

Big Data Project »
